\subsection{Detailed Description}
Header file for root finding algorhitm based on bisecting. 

Header file for root finding algorhitm based on bisection. Algorhitm works in closed domain \mbox{[}a,b\mbox{]} and there should be only one root between the domain. The range is limited to range of double.
